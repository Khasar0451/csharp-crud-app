### Backend
- ASP.NET Core
- Database managed with EF Core
- AutoMapper to handle DTOs

### Frontend
- Single Page Application
- First one in Angular (Typescript)
- Second in Blazor (C#)

### Launching
ASP.NET and Blazor with
```
dotnet run
```
Angular with 
```
ng serve
```

You can also run it in Visual Studio with .sln file
